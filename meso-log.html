🔧 Configurar MESO-LOG

Siga os passos abaixo para conectar a plataforma ao Firebase (banco de dados em nuvem gratuito do Google). Feito isso uma única vez.

1Criar projeto no Firebase

Acesse console.firebase.google.com com sua conta Google e clique em "Adicionar projeto".

Dê um nome (ex: meso-log), desative o Google Analytics se quiser, e clique em "Criar projeto".

2Ativar o Firestore

No menu lateral do projeto, clique em Build → Firestore Database → Criar banco de dados.

Escolha "Iniciar no modo de teste" (funciona por 30 dias; depois você ajusta as regras) e selecione a região southamerica-east1 (São Paulo).

3Registrar o app Web

Na página inicial do projeto, clique no ícone </> Web, dê um nome (ex: meso-log-web) e clique em "Registrar app".

Na tela seguinte aparecerá um bloco de código com firebaseConfig. Copie o objeto completo (de { até }) e cole no campo abaixo.

💡 O objeto que você precisa copiar se parece com isto:

{
  apiKey: "AIzaSy...",
  authDomain: "meso-log.firebaseapp.com",
  projectId: "meso-log",
  storageBucket: "meso-log.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
}
